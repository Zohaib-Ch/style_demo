# Component System Guidelines

## Structure
All components live in `src/components/` and are organized as:
- `ui`: Base components from shadcn UI.
- `common`: Globally reusable simple elements (containers, simple links).
- `buttons`: Premium styled interactive actions.
- `cards`: Informational card designs with premium hover effects.
- `typography`: Standardized heading and body wrappers.
- `forms`: Accessible and custom styled form components.
- `sections`: Major sections layout content.
